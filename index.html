Why isn't IOS supported?
Due to how Apple handles downloads, it is not possible to add multiple events at once as doing this would require the use of an ICS file. An ICS file is a special calendar file which allows people to share events with others conveniently. Apple limits what you can download, hence finding a way to implement this feature for Apple users isn't easy.

For now, the best I can do is let you add your events one by one using a special Google calendar link.
If you wish to add all your exams at once, or use a calendar app other than Google, try using this website on something other than an iOS device
Awesome! I got my ICS File! Now what?
In order to use your new calendar, you need to import it into whichever calendar app you use.

If you use Windows Calendar or Outlook

Great! Just double click the file and you are good to go!
If you don't have Outlook/Calendar installed and would prefer to do it online, use the guide linked below to learn how to import a ICS file to your outlook account online.

If you use Google Calendar

You will need to import the file you just downloaded into Google Calendar. Press the button below to access Google's ICS import page. From there, just upload your new ICS file and you are set!

What's an ICS?

ICS is short for iCalendar. They are plain text files that include calendar event details like a description, beginning and ending times, location, etc. Therefore, they make the perfect medium to distribute crucial dates to a large audience. Maybe VCAA could take a page from this system?
